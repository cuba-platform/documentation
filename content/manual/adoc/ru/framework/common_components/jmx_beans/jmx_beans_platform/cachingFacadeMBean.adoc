[[cachingFacadeMBean]]
====== CachingFacadeMBean

`CachingFacadeMBean` предоставляет методы очистки различных кэшей в блоках http://files.cuba-platform.com/javadoc/cuba/7.0/com/haulmont/cuba/core/jmx/CachingFacadeMBean.html[Middleware] и http://files.cuba-platform.com/javadoc/cuba/7.0/com/haulmont/cuba/web/jmx/CachingFacadeMBean.html[Web Client].

JMX ObjectName: `app-core.cuba:type=CachingFacade` и `app.cuba:type=CachingFacade`
